##  Notifyomatic 

This Project is a Location based task Remainder. 

 

## Features
 
-  Splash Screen
-  Login & Signup
-  Maps Section
-  TODO
-  Geofencing
-  Assitant
-  Feedback Section
-  Well tested Excellent UI


 
## Installation

1 ) Clone project with git

```bash
git clone https://github.com/mankotia412vishal/Notifyomatic
```
2 ) Required Android studio
